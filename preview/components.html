IT Portal Design System

프론트엔드 표준 컴포넌트

Nuxt 4 + PrimeVue + Tailwind 기반 현행 화면을 기준으로 정리한 디자인 시스템 미리보기입니다. 업무용 포털답게 밝은 서피스, 촘촘한 정보 밀도, 명확한 표 헤더, 제한된 인디고 액센트를 기본값으로 사용합니다.

Usage Rules

새 화면과 컴포넌트를 만들 때 지켜야 할 운영 규칙입니다.

  • 페이지 최상단은 PageHeader 패턴을 사용하고, 우측 액션은 2-3개 이내로 유지합니다.
  • 목록 화면은 TableCard와 StyledDataTable을 우선 사용합니다. 테이블 헤더는 blue-900, 행 hover는 zinc-50입니다.
  • StyledDataTable은 기본적으로 셀 선택/드래그/TSV 복사를 지원합니다. 편집 화면처럼 충돌 가능성이 있으면 :cell-selectable="false"로 끕니다.
  • Primary 버튼은 사용자의 다음 주요 행동 하나에만 씁니다. 보조 기능은 Secondary, 반복 도구는 아이콘 버튼으로 둡니다.
  • 도메인 색상은 KPI, 상태, 아이콘 박스 같은 작은 면적에만 씁니다. 전체 화면을 한 가지 색으로 덮지 않습니다.
  • 다이얼로그는 AppDialogHeader 패턴을 사용하고, 헤더 배경은 blue-900으로 통일합니다.
